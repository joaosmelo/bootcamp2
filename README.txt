# Bem Estar Digital Plus - Extensão Chrome

## Como usar
1. Extraia os arquivos no VS Code.
2. Abra `chrome://extensions/` no Chrome.
3. Ative **Modo do desenvolvedor**.
4. Clique em **Carregar sem compactação** e selecione a pasta deste projeto.

## Funcionalidades
- Lembretes de água, postura e alimentação em intervalos padrão (30, 20 e 120 min).
- Página de opções para personalizar os intervalos.
- Notificações com ícones diferentes.

Substitua as imagens de `icons/` por outras, se desejar.
